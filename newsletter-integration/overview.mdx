# Newsletter Integration

BeeHype supports integration with major newsletter platforms to help you automatically transform your newsletter content into engaging social media posts. Currently, we support:

- [Beehiiv](/newsletter-integration/beehiiv)
- [ConvertKit (Kit)](/newsletter-integration/kit)
- [Ghost](/newsletter-integration/ghost)
- [Substack](/newsletter-integration/substack)

## Integration Settings

You can manage your newsletter integrations at [https://www.beehype.io/app/integrations](https://www.beehype.io/app/integrations). This page allows you to:

- Connect multiple newsletter platforms
- Configure API keys and settings
- Monitor connection status
- Remove integrations when needed

## Content Synchronization

Once connected, BeeHype will:

1. Automatically detect new newsletter posts
2. Import content and media
3. Prepare it for social media transformation
4. Allow you to generate various types of social content

<Note>
For optimal performance, we recommend connecting only the newsletter platform you actively use. This ensures the most streamlined content workflow.
</Note>
