Connect your Beehiiv newsletter to automatically transform your content into engaging social media posts.

## Prerequisites

- A Beehiiv account with an active publication
- Admin access to your Beehiiv publication
- Your Beehiiv API key and Publication ID

## Getting Your API Credentials

### 1. Access API Settings

1. Log in to your Beehiiv dashboard
2. Click on "Settings" in the left sidebar at the bottom
3. Navigate to ["API"](https://app.beehiiv.com/settings/workspace/api) in the settings menu

<Frame>
  <img src="/images/newsletter-integration/beehiiv_api_setting.png" alt="Beehiiv API Settings" />
</Frame>

### 2. Generate API Key

1. In the API section, click "Generate new API key"
2. Give your key a name (e.g., "BeeHype Integration")
3. Copy the generated API key immediately - it won't be shown again

<Frame>
  <img src="/images/newsletter-integration/beehiiv_create_api_key.png" alt="Beehiiv Create API key" />
</Frame>

<Warning>
Store your API key securely. If compromised, revoke it immediately and generate a new one.
</Warning>

### 3. Find Your Publication ID

Your Publication ID is displayed in the Publication API V2 section. It starts with `pub_` followed by a string of characters.

<Frame>
  <img src="/images/newsletter-integration/beehiiv_publication_id.png" alt="Beehiiv Publication ID" />
</Frame>

## Connecting to BeeHype

1. Go to [BeeHype Integrations](https://www.beehype.io/app/integrations)
2. In the Beehiiv section, click "Connect"
3. Enter your:
   - API Key
   - Publication ID
4. Click "Save Configuration"

<Frame>
  <img src="/images/newsletter-integration/beehype_beehiiv_integration.png" alt="BeeHype Beehiiv Configuration" />
</Frame>

## Verification & Next Steps

After connecting:

1. The status will show as "Connected"
2. Go to [Create Content](https://www.beehype.io/app/create) to import your Beehiiv posts
3. Select the posts you want to transform into social media content

<Note>
You can import your posts at any time from the Create Content page. This gives you full control over which posts you want to transform into social media content.
</Note>

## Troubleshooting

If you encounter issues:

1. Verify your API key and Publication ID are correct
2. Ensure your API key has not been revoked
3. Check if your Beehiiv subscription is active
4. Confirm you have admin access to the publication

For additional help, contact [support@beehype.io](mailto:support@beehype.io)
