# Importing Content on BeeHype

BeeHype offers multiple ways to import your content and transform it into engaging social media posts. This guide will walk you through the different import options and best practices.

## Import Options

### 1. Newsletter Platforms Integration

BeeHype seamlessly integrates with major newsletter platforms:

#### Beehiiv Import
- Import directly from your Beehiiv newsletter
- Automatically imports titles, content, and formatting
- Preserves images and media
- Requires Beehiiv integration setup

#### Substack Import
- Import from your Substack publication
- Maintains original formatting and structure
- Includes images and embedded content
- Requires Substack integration setup

#### Kit Import
- Import from your Kit newsletter
- Preserves formatting and media
- Maintains content structure
- Requires Kit integration setup

### 2. Manual Content Creation

Create content directly in BeeHype:
- Write original content
- Paste from other sources
- Format using rich text editor
- Add images and media
- Apply tags for organization

## Getting Started

1. Go to [Content Creation](https://www.beehype.io/app/create)
2. Click "Import content" button
3. Choose your preferred import method:
   - Beehiiv
   - Substack
   - Kit
   - Manual input

## Troubleshooting

### Common Issues

1. **Integration Problems**
   - Verify platform connection
   - Check account permissions
   - Ensure active subscription
   - Try disconnecting and reconnecting

2. **Import Failures**
   - Check content formatting
   - Ensure stable connection
   - Try manual import if needed

3. **Content Display Issues**
   - Clear browser cache
   - Check formatting
   - Verify media links
   - Try reimporting content
